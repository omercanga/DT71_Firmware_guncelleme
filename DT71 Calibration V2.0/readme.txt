Version notes:
DT71 迷你数字镊子精确校准指引 DT71 Mini Digital Tweezers Accurate Calibration Instructions V1.11.xlsx ，Applicable to DFU V3.51~DFU V3.54
DT71 迷你数字镊子精确校准指引 DT71 Mini Digital Tweezers Accurate Calibration Instructions V2.0.xlsm ，Applicable to DFU V3.55

使用说明：						
1、DT71连接电脑进入DFU模式，将DFU磁盘中的CAL.INI文件与本文件拷贝到电脑的同一个目录下；		
2、使用Office Excel软件打开本文件，在安全警告提示中启用宏（重要!）；
3、根据实际测量结果填入相应元器件的实际值、测量值（注意单位换算）；
4、保存本文件，系统将自动生成新的CAL.INI文件；
5、将新的CAL.INI文件拷贝至DT71的DFU磁盘中覆盖原有文件，完成校准。 						
Instructions for use:									
1.Connect DT71 with computer to enters DFU mode, copy the CAL.INI file in the DFU disk and this file to the same directory on computer;
2.Open this file with Office Excel and enable macros in the security warning prompt(important!);
3.Fill in the actual values and measured values of the components according to the actual measuring (pay attention to the unit conversion);
4.Save this file, and a new CAL.INI file will be automatically generated;
5.Copy the new CAL.INI file to the DFU disk of DT71 to overwrite the original file to complete the calibration.
